pandomics documentation¶

Welcome! This website is under construction. We are trying our best to provide in depth documentation of omin but if there is something that we’ve left out please file an issue on the projects github and let us know. Report on issue on github.

pandomics is an omics-centric plugin the python software library pandas. It may or may not work with whatever version of pandas you already have installed but the only sure way to make it work is by meeting the system requirements.

If you’re development environment is fragile and you’re afraid that you will break stuff you could create a virtual env by following the directions here

Review the source code or create your own fork from the pandomics github
